# sc2Predict

## Overview

Real-time winrate prediction from starcraft 2 matches. The classification model is trained using 3464 replays released officially by ESL from the 2022/23 ESL Pro Tour. Included models are:

- Logistic Regression
- Gradient Boosting
- Random Forests
- MLP Classification
